Instructions

Follow the instructions below to get started with the game:

1. Shuffle the cards thoroughly and deal a set to each player.

2. Each player draws a card and performs the action described on it.

3. Continue this process until all cards are used or the objective is completed.

4. The player with the most points at the end of the game wins.

Make sure to read the card instructions carefully and strategize to gain an advantage.
